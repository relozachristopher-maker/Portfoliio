# Portfoliio

This repository contains a Vite + React scaffold of your portfolio, migrated from the original static site.

Quick start:

```bash
cd Reloza_NeilChristopher_MyPortfolio-react
npm install
npm run dev
```

Notes:
- Keep `style.css` at the repo root or move it to `src` and update `src/main.jsx`.
- Public assets (images, resume) are in `public/`.
- Built output is in `dist/`.
